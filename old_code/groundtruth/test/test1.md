# Year 10 Exam Booklet: Algebra and Equations

## Easy

1. Express (1/(3x-2) - 1/(3x+2)) as a single fraction.

2. Solve:
   i. x^2 + 3x + 1 = 0 (quadratic formula)
   ii. x^2 - 2x - 1 = 0 (completing the square)

3. Solve by factorising:
   i. x^2 + 4x = 0
   ii. x^2 + 11x - 26 = 0
   iii. 2x^2 - x - 6 = 0

4. Simplify: 6/(9 - x^2) - 2/(3 - x)

5. Make r the subject: x = 3r/(r+2)

6. Solve: 4v^2 + 5v - 6 = 0

7. Solve:
   i. 3/x = 4/5
   ii. 3x - 5 = (5+x)/2

8. Solve simultaneously:
   5x - 2y = -16
   3x + 4y = -7

9. Solve: 2(3 - x) ≤ 4 - x

10. Number of solutions of 2x^2 - 2x - 3?

11. Find monic quadratic with roots -4 and 7.
